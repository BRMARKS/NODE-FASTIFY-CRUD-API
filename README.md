# 🚀 Node.js + Fastify API (Versão Premium)

Projeto simples de API CRUD usando Node.js e Fastify.

## 📦 Tecnologias
- Node.js
- Fastify

## ▶️ Como rodar

```bash
npm install
npm run dev
```

Servidor roda em:
http://localhost:3000

## 📡 Rotas

- GET /items
- GET /items/:id
- POST /items
- PUT /items/:id
- DELETE /items/:id

## 💡 Exemplo POST

```json
{
  "name": "Produto 1",
  "price": 100
}
```
